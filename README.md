# Clonebot — Talk to AI Clones of Real People

Clonebot lets you have real conversations with AI-powered digital clones built from their actual books, podcasts, interviews, and keynotes. Ask questions about business, scaling, offers, lead generation, and more — and get answers in their authentic voice and style.

**Currently featuring: Alex Hormozi** — CEO of Acquisition.com, author of *$100M Offers* and *$100M Leads*.

---

## Features

- **AI Clones** — Chat with digital versions of real business leaders, trained on their actual content
- **Real-Time Responses** — Streaming answers that feel like a live conversation
- **Topic Starters** — Quick-tap topics like Offers & Pricing, Scaling, Lead Generation, Sales Strategy, and more
- **Conversation History** — Save and revisit past chats anytime
- **Server Status** — Live connection indicator so you always know the clone is ready
- **Clean Design** — Warm, intuitive interface built for focus

## Requirements

- iOS 17.0 or later
- iPhone or iPad

## Troubleshooting

| Issue | Solution |
|---|---|
| App won't connect | Check your internet connection and try again. If the server status shows "Down," wait a few minutes — it may be restarting. |
| Messages not loading | Force-close the app and reopen it. If the issue continues, check server status on the Home screen. |
| Login not working | Make sure you're using the correct email and password. Passwords are case-sensitive. |
| App crashes on launch | Update to the latest version from the App Store. If it still crashes, delete and reinstall the app. |
